# Global Scenario: User Login (Unlock)

## Context
A registered user returns to the application or reloads the page.

## Flow
1. **Reload/Launch**: User reloads the page or launches the app.
2. **Lock Screen**: User sees "Enter PIN to unlock".
2. **Input**: User enters correct 6-digit PIN.
3. **Unlock**: User clicks "Unlock" (or auto-submit).
4. **Access**: User gains access to the main dashboard.
