# Global Scenario: User Signup

## Context
A new user launches the application for the first time.

## Flow
1. **Welcome Screen**: User sees "Start Now" or similar entry point.
2. **Language Selection**: User selects preferred language (default English).
3. **Identity Creation**: User enters display name.
4. **PIN Setup**: User creates a 6-digit PIN.
5. **Confirmation**: User confirms PIN.
6. **Warning**: User acknowledges security warning (checkbox) and continues.
7. **Identity Creation**: User enters display name.
8. **Completion**: User lands on the main dashboard (Chats tab).
